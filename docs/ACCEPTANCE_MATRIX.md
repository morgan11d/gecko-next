# Матрица соответствия ТЗ

Источник требований: PDF `Задание_2_хакатон_сервис_аудио_видеоразметки_Gecko_Next.pdf`.

## Обязательный MVP

| Требование | Статус | Где проверять |
|---|---:|---|
| Загрузка аудио | Выполнено | Sidebar -> Импорт -> Аудио |
| Загрузка JSON-предразметки | Выполнено | Sidebar -> Импорт -> Аннотация |
| Waveform | Выполнено | Разметка -> медиаплеер |
| Проигрывание аудио | Выполнено | Play / Segment |
| Просмотр и редактирование сегментов | Выполнено | Список сегментов + редактор |
| Изменение start/end | Выполнено | Редактор, шаг 0.01 сек |
| Редактирование текста | Выполнено | Textarea активного сегмента |
| Автосохранение | Выполнено | Индикатор в верхней панели, localStorage |
| Экспорт JSON | Выполнено | Sidebar -> Экспорт -> JSON Gecko |
| Роли разметчика и верификатора | Выполнено | Экран входа |
| Чек-лист качества | Выполнено | Разметка -> Перед отправкой |

## Критерии приёмки

| Критерий | Вес | Покрытие |
|---|---:|---|
| Покрытие функционала Gecko | 15 | Waveform, сегменты, текст, экспорт, импорт форматов аннотаций |
| Работа с сегментами | 10 | Создание, редактирование, удаление, split/merge, границы |
| Корректность экспорта | 10 | `outputs/smoke-export.json`, `outputs/demo-export.json` |
| Автосохранение | 10 | localStorage, индикатор, восстановление после reload |
| UX/UI | 10 | Плотный рабочий интерфейс, роли, основные действия на первом экране |
| Workflow разметчик-верификатор | 10 | На проверку, Вернуть, Принять, комментарии |
| Новые функции | 10 | Термины, QA, AI mock, аналитика, версии |
| Контроль качества | 10 | Чек-лист + автоматические проверки |
| Архитектура | 10 | `docs/ARCHITECTURE.md`, типы, логика, API |
| Презентация | 5 | `docs/PRESENTATION.md`, `outputs/gecko-next-demo.webm` |

## Автотесты

- `pnpm run build` - TypeScript + production build.
- `pnpm run test:smoke` - вход, waveform, сегменты, экспорт.
- `pnpm run test:import` - импорт JSON, видимость сегментов на дорожке, редактирование и удаление.
