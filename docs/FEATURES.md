# Описание реализованного функционала

## Реализовано

- Обязательный MVP из ТЗ: загрузка аудио, JSON, waveform, проигрывание, сегменты, границы, текст, автосохранение, экспорт JSON, роли разметчика и верификатора, чек-лист.
- Расширенный MVP: видео-панель, комментарии верификатора, возврат на доработку, терминологический словарь, автоматические проверки, история изменений, горячие клавиши, статусы, аналитика.
- Дополнительные функции: mock AI-помощник, подсветка низкой уверенности ASR, QA dashboard, несколько ролей, AuditLog.

## Горячие клавиши

- `Space` - воспроизведение/пауза.
- `ArrowUp` / `ArrowDown` - переход между сегментами.
- `Ctrl+Z` / `Cmd+Z` - undo.
- `Ctrl+Y` / `Cmd+Y` - redo.
