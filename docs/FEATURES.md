# Реализованные функции и ограничения

## Реализовано

- Обязательный MVP из ТЗ: загрузка аудио, JSON, waveform, проигрывание, сегменты, границы, текст, автосохранение, экспорт JSON, роли разметчика и верификатора, чек-лист.
- Расширенный MVP: видео-панель, комментарии верификатора, возврат на доработку, терминологический словарь, автоматические проверки, история изменений, горячие клавиши, статусы, аналитика.
- Дополнительные функции: mock AI-помощник, подсветка низкой уверенности ASR, QA dashboard, несколько ролей, AuditLog.

## Горячие клавиши

- `Space` - воспроизведение/пауза.
- `ArrowUp` / `ArrowDown` - переход между сегментами.
- `Ctrl+Z` / `Cmd+Z` - undo.
- `Ctrl+Y` / `Cmd+Y` - redo.

## Известные ограничения MVP

- Нет реального backend и PostgreSQL: состояние хранится в localStorage.
- JWT/session auth показан как role-based demo login.
- AI-помощник работает как mock-логика по правилам, без внешней LLM или ASR.
- Waveform строится в браузере из аудио, серверный endpoint peaks не реализован.
- Видео синхронизируется с аудио в рамках HTML5 player, без сложной коррекции drift.
- Нет многопользовательской синхронизации в реальном времени.

## Почему ограничения допустимы для хакатона

ТЗ допускает mock для сложных ML-функций и упрощения для MVP. Главный демонстрационный сценарий закрыт: загрузка, разметка, автосохранение, проверка, возврат/приёмка, экспорт.
