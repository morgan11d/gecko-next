# Анализ gong-io/gecko

Референс: `https://github.com/gong-io/gecko`.

## Что важно из оригинального Gecko

Оригинальный Gecko описывает себя как инструмент для эффективной аннотации человеческих разговоров. По README он фокусируется на:

- сегментации голосового сигнала по спикерам;
- аннотации содержания разговора;
- редактировании результатов автоматических систем VAD, diarization, speaker identification и ASR;
- визуализации нескольких источников аннотаций одновременно;
- сравнении разных файлов аннотаций;
- работе без серверной части;
- форматах RTTM, CTM, JSON, CSV;
- ускорении работы горячими клавишами;
- WaveSurfer.js как аудио-движке.

Важно: сам репозиторий помечен как deprecated с ноября 2025 года, поэтому мы используем его как исторический UX/функциональный референс, а не как техническую базу для production.

## Что перенесено в Gecko Next MVP

| Идея Gecko | Реализация в MVP |
|---|---|
| WaveSurfer waveform | React + WaveSurfer.js |
| Редактирование ASR-предразметки | Импорт JSON/CSV/RTTM/CTM и правка сегментов |
| Сегментация по времени | Segment rail + start/end editor |
| Диаризация / спикеры | Speaker field, original speaker names, protected speaker rename |
| Несколько источников | Две дорожки `current` и `source` на таймлайне |
| Сравнение аннотаций | Baseline/source rail + history/version restore |
| Горячие клавиши | Space, ArrowUp/Down, Ctrl/Cmd+Z/Y |
| Работа без сервера | Статический MVP, localStorage |
| Экспорт разметки | Gecko-compatible JSON + QA report |

## Что добавлено сверх оригинального Gecko

- Workflow разметчик -> верификатор -> принято/доработка.
- Чек-листы качества.
- Автосохранение с индикатором.
- Терминологический модуль.
- Mock AI-помощник.
- Аналитика по качеству и прогрессу.
- AuditLog.
- Поддержка видео-панели.

## Вывод

Наш MVP закрывает базовые Gecko-паттерны и расширяет их требованиями PDF: контролем качества, терминологией, верификацией, аналитикой, AI-подсказками и промышленным workflow.
