# Краткий план развития после хакатона

## 1. Довести backend до production

- Перенести доменную модель в PostgreSQL.
- Подключить SQLAlchemy repository к существующей FastAPI-архитектуре.
- Реализовать JWT/session auth, refresh tokens и RBAC.
- Перенести AuditLog и version history в БД.

## 2. Усилить работу с медиа

- Серверная генерация peaks для файлов до 60 минут и больше.
- Ленивое отображение больших аудио.
- Snap-to-speech и подсказки границ.

## 3. Расширить командный workflow

- Очередь задач и массовое назначение.
- Уведомления о возвратах.
- SLA, дедлайны, приоритеты.

## 4. Подключить AI/ASR

- Интеграция Whisper/Vosk или внешнего ASR.
- Поиск пропущенной речи.
- Поиск обрезанных слов.
- Терминологические подсказки через LLM.

## 5. Подготовить DataOps

- Версионирование датасетов.
- Экспорт в Gecko JSON, JSONL, CSV, SRT/VTT.
- Отчёты по качеству для заказчика.
- Интеграция с ML/ASR пайплайном.
